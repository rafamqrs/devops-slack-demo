# Fase 2: O Caso de Negócio no Salesforce

**Objetivo:** Ter um código real para ser "analisado" pelo Agentforce e enviado via PR.

**Instruções de Geração:**
1. Crie uma estrutura SFDX padrão (`force-app/main/default`).
2. Gere um componente LWC `churnRiskIndicator` com um ficheiro `.js`, `.html` e `.js-meta.xml` (exposto na Record Page de Opportunity).
3. Gere a classe Apex `ChurnRiskController.cls` e a sua respetiva classe de teste `ChurnRiskControllerTest.cls` com 100% de cobertura.
4. O código deve ter comentários claros e seguir boas práticas (o "Agentforce" na narrativa da demo vai elogiar isso).
